A provider class that manages a list of \mbox{[}\doxylink{class_station}{Station}\mbox{]} instances.

This class uses the \mbox{[}\doxylink{class_change_notifier}{Change\+Notifier}\mbox{]} class from the Flutter framework to notify listeners of changes to the list of stations. It also provides a method for fetching stations from the \mbox{[}\doxylink{class_station_service}{Station\+Service}\mbox{]} and a method for getting a list of station names.
