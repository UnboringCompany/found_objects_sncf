library}

A widget that allows users to search for lost objects.

This widget provides text fields for entering the station name and object type, as well as date pickers for selecting the start and end dates. It uses the \mbox{[}Objects\+Provider\mbox{]} and \mbox{[}Station\+Provider\mbox{]} classes to fetch objects and stations, respectively. It also uses the \mbox{[}Object\+Tile\mbox{]} widget to display search results.
